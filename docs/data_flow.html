How Engines Connect
Engine 1 WRITES → :CrowdSignal nodes to Neo4j (every 60s from live chat)
Engine 2 READS ← :CrowdSignal + :ExternalFactor + :User from Neo4j → generates alert
Engine 3 READS ← :CrowdSignal (detects "cancel") + :Train (next 3 trains) → cascade math → reroute
Engine 4 READS ← :User context from Neo4j (skips cold-start) + legal docs from Milvus → agent decides tools
Feedback: E4 complaints → enrich E1 safety heatmap. E1 signals → feed E2 + E3 predictions.
Feedback: E3 disruption events → stored as historical nodes → improve future cascade predictions.
